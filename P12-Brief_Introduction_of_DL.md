如果一个神经网络的参数已知，那么它其实就是一个函数，它接受向量作为输入，输出另一个向量。

如果它的参数还位置，但是结构已知，那么它就是一个 Function Set. 当然这个 Function Set 是很大的，包含了单层的线性回归或逻辑回归都无法拟合的函数。
